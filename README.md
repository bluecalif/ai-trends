# AI Trend Monitor

RSS 기반 AI 기술 트렌드 모니터링 서비스

## 개요

AI 기술/산업/연구 트렌드를 분야·사건 단위로 구조화하여 한눈에 파악할 수 있는 개인용 모니터링 서비스입니다.

## 주요 기능

- RSS 피드 수집 및 정규화
- AI 기반 요약 및 엔티티 추출
- IPTC/IAB 표준 분류 + 커스텀 AI 태그
- 중복/사건 그룹화 및 타임라인 구성
- 인물별 사업/기술 방향 추적

## 기술 스택

- Backend: Python (FastAPI), PostgreSQL
- Frontend: Next.js 14+ (TypeScript, Tailwind CSS)
- AI: OpenAI API
